BetAI WERSJA 1807 — Ranking Premium Minimal

Zmiana wyłącznie wizualna w tabeli zakładki Ranking:
- cienkie kolorowe podkreślenia nagłówków statystyk,
- subtelne kolorystyczne rozdzielenie kolumn,
- delikatny efekt hover,
- bez zmian logiki, danych, Supabase, rankingu i przycisków.

Wdrożenie: wgraj folder dist na Netlify albo podłącz cały projekt i uruchom npm install + npm run build.
SQL z wersji 1806 pozostaje dołączony tylko jako kopia poprzedniej naprawy; dla samego wyglądu 1807 nie trzeba uruchamiać nowego SQL.
