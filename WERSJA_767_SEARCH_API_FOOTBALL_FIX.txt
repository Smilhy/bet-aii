WERSJA 767 — naprawa wyszukiwarki w Dodaj typ

Zmiany:
1. Dla mode=search backend omija The Odds API jako główne źródło listy i od razu używa API-FOOTBALL /teams + /fixtures.
2. Dzięki temu frazy typu "Legia" są wyszukiwane globalnie po drużynie, a nie tylko w krótkiej liście meczów zwróconych przez The Odds API.
3. Usunięto podwójne odpalanie wyszukiwania po Enterze w formularzu Dodaj typ.
4. Kursy nadal są dociągane dla znalezionych meczów przez istniejącą logikę wzbogacania wyników.

Uwaga:
Jeżeli dzienny limit API-FOOTBALL jest przekroczony, wyszukiwarka może dalej zwracać puste wyniki do czasu odnowienia limitu.
