WERSJA 1853 — DODAJ TYP / ADD PICK — pełne tłumaczenie EN

Co poprawiono:
- przetłumaczono ekran „Dodaj typ” po przełączeniu języka na angielski,
- przetłumaczono hero, wyszukiwarkę, popularne sporty, tryb auto/manual,
- przetłumaczono listę meczów, puste stany, panel rynków i panel kuponu,
- przetłumaczono formularz ręczny i komunikaty/toasty związane z dodawaniem typu,
- dodano szersze tłumaczenia nazw sportów i krajów w sidebarze Add pick,
- dodano dodatkowe reguły tłumaczeń dla dynamicznych etykiet (np. liczba meczów, remaining today, total odds panel).

Build:
- wykonano npm install
- wykonano npm run build
